CoolTip - jQuery Tool Tip Plugin

Plugin Overview

This plugin was made to easily allow the use of a simple JavaScript code to initialize any element you wish to show a tool tip, that's surrounded by an anchor tag. In the next section, the examples will show what it can do, then the specs and so on...

Plugin Specifications

In order to use the plugin, you need jQuery, which can be obtained by going to their website. You also need the plugin itself, the stylesheet, and the arrow image to make it look more like a bubble. Here are the specs:

How to Use It

First off, you need to include jQuery, the plugin, and the CSS

<link rel="stylesheet" type="text/css" href="cooltip.css">
<script type="text/javascript" src="jquery-1.7.1.min.js"></script>
<script type="text/javascript" src="cooltip.jquery.js"></script>

Now, come up with a class name, id, or just use the element if you want that element to always have a tool tip, and create the JavaScript for it

<script type="text/javascript">
    $(function() {
        $(".ct").cooltip(); // initialize the plugin with this class
    });
</script>

And finally just create the element

<a href="http://www.example.com" class="ct" title="This is where the tool tip text will go">A Link</a>
